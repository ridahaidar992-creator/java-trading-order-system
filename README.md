
# Trading Order System

A Java console-based trading order management system.

## Features
- Create orders
- View all orders
- Search order by ID
- Update order
- Delete order
- Save/load orders from file
- Show statistics
- Sort orders by price
- Export orders to CSV
- Portfolio summary

## Technologies Used
- Java
- Object-Oriented Programming
- ArrayList
- File handling
- Packages

## Project Structure
- `app` → main application entry
- `model` → order data model
- `service` → business logic and file handling

## How to Run
Run `Main.java` from the `app` package.
